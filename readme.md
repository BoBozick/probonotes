Probonotes (_pro bono publico_, for the public good, or _pro Bo's notes_, for my notes) is a Typst styling package for mathematical notes with theorem environments and mathematical notation.

Note that this package is not published to the offical repository, so it must be installed locally.

For an example, see [my linear algebra notes](https://github.com/BoBozick/linalg-notes).

Imported packages: TheOrion, Physica.
